# prevep example

This is an example to illustrate the use of `prevep`, a module to load data from other games, such as a previous episode in a game series. More information about this module can be found [here](https://github.com/castro67/renpyglossa/tree/main/libs/prevep).

The example is a typical new project created by the Ren'Py Launcher with very few additions. It can be built with Ren'Py just like any other Ren'Py project.

We'll imagine that this example is the second episode of a game series whose first episode is "The Question", one of the sample projects included in the Ren'Py SDK. 

Three files are relevant to this example:

* [prevep.rpy](game/libs/prevep.rpy): the module whose use we want to illustrate.
* [script.rpy](game/script.rpy): where data from another game is managed and some explanations are given.
* [screens.rpy](game/screens.rpy): this is a file generated by Ren'Py where only one line has been added, so we can access the load screen in `prevep`:

        textbutton _("Episode 1") action ShowMenu("prevep_slots", "the_question-7", _("Continue from \"The Question\""))
