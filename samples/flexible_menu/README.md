# `flexible_menu` example

This is an example to illustrate the use of `flexible_menu()`, a function to display menus, where choices can be enabled, disabled or hidden regardless of the value of `config.menu_include_disabled`. More information about this function can be found [here](https://github.com/castro67/renpyglossa/tree/main/libs/flexible_menu).

The example is a typical new project created by the Ren'Py Launcher with very few additions. It can be built with Ren'Py just like any other Ren'Py project.

Only two files are relevant to this example:

* [flexible_menu.rpy](game/libs/flexible_menu.rpy): the function whose use we want to illustrate.
* [script.rpy](game/script.rpy): where that function is called to explain the problem it can help solve.
