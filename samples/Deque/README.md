# Deque example

This is an example to illustrate the use of the `Deque`, a revertible double-ended queue for Ren'Py. More information about this class can be found [here](../../libs/Deque).

The example is a typical new project created by the Ren'Py Launcher with very few additions. It can be built with Ren'Py just like any other Ren'Py project.

Only three files are relevant to this example:

* [Deque.rpy](game/libs/Deque.rpy): the class whose use we want to illustrate.
* [script.rpy](game/script.rpy): where an object of that class is created and used, and some explanations are given.
* [test_screen.rpy](game/test_screen.rpy): where the user can choose what values are appended to the queue.
